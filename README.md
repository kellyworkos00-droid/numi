# numi-kicks

A minimal Node.js + TypeScript starter project.

## Requirements

- Node.js 20+

## Scripts

- `npm run dev`: Run in watch mode with TS execution
- `npm run build`: Compile TypeScript to `dist/`
- `npm start`: Run compiled output
- `npm test`: Execute tests via Vitest

## Quick Start

```bash
npm install
npm run build
npm start
```
